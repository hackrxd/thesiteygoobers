Anonymous Anomaly

99 Days, 99 Minutes, 99 Seconds, 99 Seconds

The Anonymous Anomaly is an event where, each day at midnight UTC, a user is chosen. The user will recieve a DM (Direct Message), where the bot sends a message asking them to say something to be posted for the event. The event will last 11 days and start on October 1st. The server's rules do apply, and any violation will result in hackr checking who made it and punishing them as needed.
